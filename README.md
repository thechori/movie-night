# movieNight
Web application to track and organize Sunday Movie Nights

## Installation
- Install Meteor

```
$ curl https://install.meteor.com | sh
``` 

- Grab the code
```
$ git clone git@github.com:thechori/movie-night.git
```

- Navigate into the directory
```
$ cd movie-night
```

- Run the application
```
meteor
```
